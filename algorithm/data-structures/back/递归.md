
## 递归
**递归**：
* 若一个对象部分地包含自己，或用自己做给自己定义，则称该对象为递归
* 若一个过程直接地或间接地调用自己，则称该过程为递归过程
例如熟悉的阶乘函数：
$$
Fact(n)=
\begin{cases}
1, & 若 n=0\\[5ex]
n * Fact(n-1), & n>0
\end{cases}
$$

常见的程序代码：
```
long fact(参数){
    if(递归结束条件) 可直接求解步骤; // 基础项
    else fact(较小参数); // 归纳项
}
```
递归求n的阶乘：
```
long fact(long n){
    if(n==0) return 1;
    else return n * fact(n-1);
}
```

**常用到递归的情况**：
1. 递归定义的数学函数
1. 具有递归特性的数据结构
1. 可递归求解的问题
1. 汉诺塔问题

**优点**：结构清晰，程序易读

**缺点**：每次调用要生成新的存储空间，保存状态信息，时间开销大
